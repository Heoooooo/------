form

폼과 관련된 작업들은 정보 저장 목정, 정보 검색, 정보 수정 등으로 데이터베이스 기반이며
입력된 정보처리를 위해 스크립트 언어 또는 서버 프로그래밍을 사용한다.

HTML Form Elements
lavel : 각 입력 양식 요소를 위함
textarea : 텍스트 영역
input : 기본 입력 양식 요소
fieldset : 폼 요소를 그룹으로 묶는 태그
legend : 그룹으로 묶는 구역에 제목을 붙이는 태그
datalist : 데이터 목록에 제시한 값 중 선택하면 텍스트 필드에 입력
select : 드롭다운 목록(=콤보박스), option, optgroup
button : 다양한 형태의 버튼 삽입
output : 계산 결과

input태그의 다양한 속성

뭐가 있을까나
value : 디폴트 값
disabled : 클릭 안되게 설정, 사용 안됨, 폼 요소로 제출 안됨
readonly : 내용을 보기만 하고 입력하지 못하게 함
required : 필수 필드 체크, 필수 필드는 브라우저에서 직접 체크하는 것으로 오류 메시지 내용은 브라우저마다 다르게 나타남
size : 보여지는 너비 설정(문자수), 넘겨도 되지만 보이지 않음
min, max : 타입에 최소, 최대 값 설정(number, range, date, month, time, week)
multiple : 다중선택허용(file, select)
placeholder : 입력란에 표시하는 힌트로, 필드를 클릭하면 사라짐
pattern : 입력 값 체크
autofocus : 페이지를 불러오자마자 원하는 폼 요소에 마우스 커서 표시
autocomplete : 입력 요소에 들어갈 값을 예측해서 보여줌
