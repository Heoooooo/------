CSS 기본 선택자

선택자 종류
전체 선택자 *{} : 페이지에 있는 모든 요소를 대상으로 스타일을 적용할때 사용하므로 주로 초기화할때 사용
태그 선택자 태그명{} : 특정 태그를 사용한 모든 요소에 스타일이 적용
클래스 선택자 .클래스명{} : 요소의 특정 부분에만 스타일 적용, 클래스니까 보통 여러번 반복할 스타일에 적용
아이디 선택자 #아이디명{} : 요소의 특정 부분에만 스타일 적용, 아이디ㅣ까 문서 안에서 한번만 사용될 스타일

속성 선택자도 있으며, 그것은 형식이 값에관련하여 다양하다.

그룹 선택자도 있는데, 그것은 같은 스타일을 사용하는 선택자를 한번에 정의한다.

연결 선택자

선택자 종류
하위 선택자 상위요소 하위요소{} : 부모 요소에 포함된 모든 하위 요소에 스타일 적용
자식 선택자 부모요소>자식요소{} : 자식 요소에 스타일을 적용하는 선택자
인접 형제 선택자 요소1 + 요소{} : 같은 부모를 가진 형제 요소 중 첫번째 동생 요소에만 스타일 적용
형제 선택자 요소1~요소2{} : 형제 요소들에 스타일 적용

가상 클래스 선택자

특정 상태에 따라 스타일을 정의할 때 사용

선택자 종류
링크 선택자(사용자의 동작에 반응) : link, visited, active, hover
상태 선택자 : checked, focus, enabld, disabled(여기까지 모두 input태그 선택), target(앵커로 연결된 부분에 스타일 적용)
not 선택자 not(선택자) : 특정 요소를 제외하고 스타일 적용
구조 선택자 : first-child, last-child, nth-child, nth-last-child. 그리고 그 사이에 모두 of를 붙인것들
가상 요소 : ::first-line, ::first-letter, ::before, ::after

박스

웹 페이지의 레이아웃 구성에 사용 : margin, padding, border, width, height

구체적으로 어떻게?
margin : 크기or백분율or auto: 요소와 요소 간의 간격 조절 가능
border : 테두리
border-width : 테두리 두께, border-style : 테두리 형태(너무다양...), border-color : 테두리 색상
이것들 다 묶어서 표기 가능 *테두리 형태에서 우리가 알아야할거 none, hidden, solid, dotted,dashed,double..
border-radius : 박스 모델의 테두리를 둥글게 처리 크기 또는 백분율 ->50%하면 원->모서리마다 다르게 처리가능
padding : 콘텐츠와 테두리 사이의 여백
width, height : 콘텐츠의 너비, 높이
box-sizing : boder-box(테두리까지 포함하여 너빗값을 지정)
content-box(콘텐츠 영역만 너비값을 지정)
box-shadow : 선택한 요소에 그림자 효과 내기, 가로 세로 흐림정도 번짐정도 색상
